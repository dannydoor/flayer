# flayer
국군 장병들이 편하게 뮤직비디오를 감상할 수 있도록 제공하는 뮤직비디오 플레이어입니다.
*크롬*에서 가장 정상적으로 구동할 수 있으며, 다른 브라우저, 특히 인터넷 익스플로러에서는 몇몇 기능과 UI가 비정상적일 수도 있습니다.
자세한 사용법은 flayer 내의 도움말을 참고해주세요
